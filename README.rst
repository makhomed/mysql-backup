========
mysql-backup
========

MySQL backup tool

Installation
------------

- ``cd /opt``
- ``git clone https://github.com/makhomed/mysql-backup.git``

Upgrade
-------

- ``cd /opt/mysql-backup``
- ``git pull``


Usage
-----

.. code-block:: none

    # mkdir /srv/mysql-backup
    # /opt/mysql-backup/mysql-backup
